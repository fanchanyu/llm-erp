## 💡 學到的事 / 發現

```
1. ERP 三層架構：戰略層(MPS/MRP/CRP) → 戰術層(APS/CAPP) → 執行層(MES/IoT)
2. 動態重排程策略：右移(Right-Shift) vs 改變路徑(Route Changing)
3. BOM 多階展開：毛需求→扣除庫存→淨需求，逐層向下
4. APS 演算法：MILP(精確) / GA/SA/PSO(啟發式) / RL(學習式)
5. LLM-ERP 設計要支援多 Provider，不要鎖單一 LLM
6. CrossRef DOI 驗證：curl doi.org 確認 HTTP 302 即可驗證出版真實性
7. Springer 書籍（如 Pinedo Scheduling）可透過 DOI 驗證
8. SQLAlchemy Uuid 類型可跨 PostgreSQL/SQLite 使用
9. Docker Desktop 未安裝，開發先用 SQLite 零依賴
```
